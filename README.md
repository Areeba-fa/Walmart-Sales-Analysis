# 📊 Walmart Sales Predictive Analytics
A comprehensive 6-week data science project completed during my internship at **@Afynix Digital**.

## 🛠️ Tech Stack
- **Python** (Pandas, Scikit-Learn)
- **SQL** (SQLite3)
- **Dashboarding** (Streamlit, Plotly)

## 📈 Key Achievements
- Built a **Machine Learning** model to predict weekly revenue.
- Quantified the impact of holidays ($69k+ lift) and unemployment risks.
- Developed an interactive dashboard for executive decision-making.

## 📁 Project Structure
- `week5_machine_learning.py`: Predictive Engine & ML Model.
- `dashboard.py`: Interactive Streamlit interface.
- `week3_queries.py`: SQL Database migration & benchmarking.
- `cleaning.py`: Data preprocessing & integrity checks.
